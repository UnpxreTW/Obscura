# Obscura

純 Swift 從零打造的作業系統 —— 繁體中文學習筆記。

@Metadata {
    @TechnologyRoot

名字取自 camera obscura（暗箱）：用一個小孔把外界世界投影進暗箱裡供人觀察——這也是
專案裡模擬器的精神，把龐大的硬體世界投影成一幅能反覆端詳、慢慢看懂的影像。

## Overview

一場純 Swift 的實驗：從零寫一個作業系統，連用來跑它的模擬器都自己用 Swift 寫，看這
語言能往系統底層走多遠。整個過程以繁體中文記錄，是個人學習筆記、不是教材。
